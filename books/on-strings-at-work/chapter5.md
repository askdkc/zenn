ミナは前章で見た申請メールを、Emacsの一つのバッファに貼り付けた。Excelに入れる前の、仮置き場である。

バッファには、挨拶、署名、備考、必要項目が混ざっていた。

```text
お世話になっております。
申請番号: A-1029
氏名: 山田　太郎
部署: 営業一課
金額: 12,000円
備考: 急ぎ

いつもお世話になっております。
申請番号: A-1030
名前: 佐藤 花子
部門: 営業二課
金額: 12000
備考: 確認してください
```

「このままだと、読むだけで怠ぃです。まず必要な行だけ見たい。」

「検索？」

「検索だけだと、一件ずつ飛ぶだけです。今欲しいのは、必要な行だけを一度に残す作業です。」

## 4.1 必要な行だけ見る `keep-lines`

ミナはまず、必要な項目が含まれていそうな行だけを残すことにした。

「申請番号、氏名、名前、部署、部門、金額。この辺が入ってる行だけ見たいんですよ。」

ミナは式を入力した。

```elisp
(keep-lines "申請番号\\|氏名\\|名前\\|部署\\|部門\\|金額")
```

高村はすぐに眉を寄せた。

「ごめん、その `\\|` って何？」

ミナは、式を分解してホワイトボードに書いた。

```text
申請番号\\|氏名\\|名前\\|部署\\|部門\\|金額
```

「これはEmacsの正規表現です。正規表現って名前は怖いですけど、ここでは『または』の記号だけ覚えれば大丈夫です。`\\|` は『または』という意味です。」

```text
申請番号\\|氏名
  = 「申請番号」または「氏名」を含む行
```

つまり、この式はこう読める。

```text
申請番号 または 氏名 または 名前 または 部署 または 部門 または 金額
を含む行だけ残す。
```

## 入力例

```text
お世話になっております。
申請番号: A-1029
氏名: 山田　太郎
部署: 営業一課
金額: 12,000円
備考: 急ぎ
```

## 出力例

```text
申請番号: A-1029
氏名: 山田　太郎
部署: 営業一課
金額: 12,000円
```

ここで、長いメールは「必要そうな行のリスト」になった。

```text
長いメール本文
  ↓ keep-lines
必要そうな行のリスト
```

高村は画面を見た。

「文章が、表になる前の材料になった感じだね。」

「そうです。文章をそのまま表に押し込むんじゃなくて、行のリストにほどいてから扱います。」

## 4.2 不要な行を消す `flush-lines`

必要な行だけを残す方法は分かった。だが、ミナは別のメールを見て、少し首を傾げた。

挨拶や署名が多すぎる。必要な行を残すより、まず邪魔な定型文だけ消した方が早い場合もある。

「お世話になっております、よろしくお願いします、みたいな行が多すぎますね。こういうのは業務的には大事ですけど、データ化するときはノイズです。」

高村が言った。

「人間関係には必要だけど、DBには不要ってことか。」

「そうです。人間には礼儀、機械にはノイズです。」

ミナは不要な行を消す式を書いた。

```elisp
(flush-lines "お世話になっております\\|よろしくお願いいたします")
```

`flush-lines` は、条件に合う行を消す。  
`keep-lines` が残すなら、`flush-lines` は消す。

ただし、問題もある。

```text
備考: よろしくお願いいたします。急ぎです。
```

この行は、備考として意味がある。単純に「よろしくお願いいたします」を含む行を消すと、この備考まで消える可能性がある。

そこで、行全体が定型文だけの場合に限定する。

```elisp
(flush-lines "^よろしくお願いいたします。$")
```

ここで出てくる `^` と `$` は、正規表現の目印である。

```text
^
  行の先頭

$
  行の終わり
```

つまり、この式は「行の先頭から行の終わりまで、内容が `よろしくお願いいたします。` だけの行」を消す。

```text
消える:
  よろしくお願いいたします。

消えない:
  備考: よろしくお願いいたします。急ぎです。
```

## ミナのメモ

ミナは右下のメモバッファに、ここまでの変換を整理した。

```text
現実:
  メール本文が長く、必要情報が埋もれている。

Emacsでやったこと:
  keep-lines で必要行を残す。
  flush-lines で不要行を消す。

抽象化:
  長文メール
    ↓
  必要行のリスト

次へ:
  必要行を、項目名と値のペアへ分ける。
```
